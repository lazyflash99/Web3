Demo Day Setup Guide

Everything you need to try the Gas Fee Optimizer live demo — from zero to running in 5 minutes

Beginner Friendly Sepolia Testnet No Real Money Needed

Before You Start — Checklist

  • A laptop/desktop with Google Chrome or Brave browser
  • An internet connection
  • That's it! Everything else we'll set up below.
We use the Sepolia testnet — a practice version of Ethereum. All the ETH here is fake/free. You will never spend real money.

Part 1: Install MetaMask Wallet

What is MetaMask?

MetaMask is a browser extension that acts as your crypto wallet. Think of it like a digital wallet app (Paytm/GPay) but for Ethereum.

1
Open Chrome and go to the MetaMask website
https://metamask.io/download/

Click "Install MetaMask for Chrome". This opens the Chrome Web Store.

2
Add the extension

Click "Add to Chrome" → "Add extension" in the popup.

You'll see a fox icon appear in your browser toolbar (top-right). If you don't see it, click the puzzle icon and pin MetaMask.

3
Create a new wallet
  1. MetaMask opens automatically. Click "Create a new wallet".
  2. Agree to the terms.
  3. Set a password (at least 8 characters). This is just for your browser — pick something simple for demo day.
  4. It will show you a Secret Recovery Phrase (12 words). For a testnet demo, you can skip backing it up — click "Remind me later" if given the option. For real wallets, ALWAYS save this phrase.
  5. Click "Done" or "Got it".
This is a TESTNET-only wallet. Don't send real ETH to it. For this demo, the recovery phrase doesn't matter since we're using fake ETH.
MetaMask is installed! You should see a wallet with 0 ETH on Ethereum Mainnet. Next, we switch to Sepolia.

Part 2: Switch to Sepolia Testnet

1
Open MetaMask

Click the fox icon in your browser toolbar.

2
Enable test networks
  1. Click the network dropdown at the top (it says "Ethereum Mainnet").
  2. If you see "Sepolia" listed, skip to step 3.
  3. If not: Click the three dots or "Show test networks" toggle at the bottom of the dropdown, or go to Settings → Advanced → Show test networks → ON.
3
Select Sepolia

Click "Sepolia test network" in the dropdown. The network name at the top should now say Sepolia.

Your balance will show 0 SepoliaETH — that's expected. We'll get free ETH next.

You're now on Sepolia! The network indicator should show "Sepolia" with a teal/blue color.

Part 4: Run the Live Demo

1
Open the Gas Fee Optimizer
LIVE DEMO
https://ask-812.github.io/Web3/
2
Connect your wallet

Click the "Connect Wallet" button. MetaMask will pop up asking you to connect — click "Next" then "Connect".

If MetaMask asks you to switch to Sepolia, click "Switch network". The website auto-detects the correct network.
3
Load demo actions

Click "Quick Demo (5 Actions)". This loads 5 sample blockchain transactions into the action list.

You'll see something like:

Action 1: setValue(42) ~75,000 gas Action 2: increment() ~75,000 gas Action 3: setValue(100) ~75,000 gas Action 4: increment() ~75,000 gas Action 5: setValue(999) ~75,000 gas --- Total if done individually: ~375,000 gas
4
Choose an execution mode

You'll see 4 buttons. Here's what each does:

ModeWhat It DoesNeeds ETH?
Execute IndividuallySends 5 separate transactions (the old/expensive way)Yes
Standard BatchBundles all 5 into 1 transaction (saves gas!)Yes
Compressed BatchBundles + compresses calldata (saves even more gas!)Yes
Meta-Tx via RelayerGasless! You sign a message and a relayer pays the gas for youNo!
5
Try "Standard Batch" first
  1. Click "Standard Batch".
  2. MetaMask pops up showing the transaction. Click "Confirm".
  3. Wait for the transaction to be mined (10-30 seconds on Sepolia).
  4. The Activity Log will show gas used and savings compared to individual execution.
6
Try the other modes

Reload the page, reconnect, load 5 actions again, and try "Compressed Batch" or "Meta-Tx via Relayer".

Compare the gas usage in the Activity Log!

Expected Results:
- Individual: 5 transactions, ~375,000 total gas
- Standard Batch: 1 transaction, ~270,000 gas (saves ~28%)
- Compressed Batch: 1 transaction, ~250,000 gas (saves ~33%)
- Meta-Tx: 0 gas for you! The relayer pays.

No ETH? Use Gasless Mode!

If faucets are down or you don't have any SepoliaETH, you can still demo the project:

  1. Open the demo site and connect MetaMask (make sure you're on Sepolia).
  2. Click "Quick Demo (5 Actions)".
  3. Click "Meta-Tx via Relayer".
  4. MetaMask will ask you to sign a message (NOT a transaction). This is free — no gas needed.
  5. The relayer server pays the gas on your behalf and submits the transaction.
  6. You'll see the result in the Activity Log.
This is the gasless/meta-transaction feature — one of the key innovations of this project. Users sign an EIP-712 message, and a relayer submits the transaction and pays the gas fee.

Troubleshooting

MetaMask doesn't pop up when I click "Connect Wallet"

  • Make sure MetaMask is installed and the fox icon is visible in your toolbar.
  • Try clicking the fox icon first to unlock MetaMask (enter your password).
  • Refresh the page and try again.

"Wrong network" or "Switch to Sepolia" errors

  • Open MetaMask → click the network dropdown at the top → select Sepolia test network.
  • The website should also prompt you to switch automatically.

Transaction fails or "insufficient funds"

  • Check your SepoliaETH balance in MetaMask. You need at least 0.005 ETH.
  • Try a different faucet from Part 3 above.
  • Or use "Meta-Tx via Relayer" which needs zero ETH.

"Rate limited" or slow loading

  • The website uses free public RPCs which have rate limits. Just wait a few seconds and try again.
  • The website automatically rotates between 5 different RPC endpoints.
  • If batch modes fail, the website falls back to the relayer automatically.

MetaMask shows a warning/alert on the transaction

  • Click "Review alert" or "I want to proceed anyway".
  • Then click "Confirm".
  • This is normal for testnet contracts — MetaMask warns about unverified contracts.

Nothing happens after confirming the transaction

  • Sepolia can be slow sometimes. Wait up to 30 seconds.
  • Check the Activity Log on the page for status updates.
  • Open MetaMask and check the "Activity" tab to see if the transaction is pending.

Key Concepts (1-liner each)

TermWhat It Means
GasThe fee you pay for any Ethereum transaction (like a shipping charge).
Batch TransactionCombining multiple actions into one transaction to save on base fees.
Meta-TransactionYou sign a message; someone else submits and pays the gas for you.
RelayerA server that receives your signed message and submits the transaction on your behalf.
SepoliaA free test version of Ethereum for developers. No real money involved.
MetaMaskA browser-based crypto wallet that connects websites to Ethereum.
Calldata CompressionMaking transaction data smaller so it costs less gas to store on-chain.
