Step-by-Step Deployment

1
Get Sepolia ETH

Go to sepoliafaucet.com or Alchemy Sepolia Faucet and request test ETH for your deployment wallet. You need ~0.05 ETH.

2
Get an Alchemy/Infura API Key

Create a free account at alchemy.com, create an app on Sepolia, and copy the HTTPS URL.

3
Configure .env
cp .env.example .env # Edit .env with your values: PRIVATE_KEY=your_wallet_private_key_without_0x SEPOLIA_RPC_URL=https://eth-sepolia.g.alchemy.com/v2/YOUR_KEY ETHERSCAN_API_KEY=your_etherscan_api_key
Warning: Never commit your .env file to Git! It's already in .gitignore.
4
Deploy to Sepolia
npm run deploy:sepolia

This will deploy all 7 contracts, verify them on Etherscan, and generate hosted/index.html with embedded addresses.

5
Host the Frontend

Upload hosted/index.html to any free static host:

Option A: Vercel (Recommended — 30 seconds)

  1. Go to vercel.com → Sign up with GitHub
  2. Click "Add New Project" → "Import Git Repository" or drag-drop the hosted/ folder
  3. Click Deploy → Get your live URL

Option B: Netlify (Drag & Drop)

  1. Go to app.netlify.com/drop
  2. Drag the hosted/ folder onto the page
  3. Instant URL generated

Option C: GitHub Pages

  1. Push hosted/index.html to a GitHub repo
  2. Go to Settings → Pages → Deploy from branch
  3. URL: https://username.github.io/repo-name
6
Fill in links above & export this page as PDF

After deploying, fill in the contract addresses and frontend link above, then print this page as PDF (Ctrl+P → Save as PDF).

🧪 Testing the Deployed Frontend

  1. Open the hosted URL in Chrome with MetaMask installed
  2. Switch MetaMask to Sepolia testnet
  3. Click "Connect Wallet" → "Load Contracts"
  4. Click "Quick Demo (5 Actions)" to load test actions
  5. Click "Execute Individually" first (baseline), then reload and try "Standard Batch" or "Compressed Batch"
  6. Compare gas usage in the bar chart
